**Shasn Online** is a web-based adaptation of the popular political strategy board game **Shasn**, bringing its core mechanics to an interactive multiplayer experience. The project aims to faithfully recreate the strategic depth of the original game while enabling real-time online gameplay, turn management, and seamless player interaction.
